# AWSP - AWS Profile Switcher in Go

Easily switch between AWS Profiles

## Prerequisite
- [Go(1.21)](https://golang.org/dl/)

## Setup
```sh
make install
```

Add the following to your .bashrc or .zshrc config
```sh
alias awsp="source _awsp"
```

## Usage
```sh
awsp
```
